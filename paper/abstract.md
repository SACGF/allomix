# allomix: donor chimerism monitoring from existing clinical NGS panels

## Abstract

Many clinical next-generation sequencing (NGS) panels already include biallelic polymorphic markers for sample identification or quality control. We reasoned that these markers could be repurposed for quantitative donor chimerism monitoring after allogeneic hematopoietic stem cell transplantation (HSCT), avoiding the need for a separate dedicated assay. We present allomix, an open-source tool that estimates single-donor or multi-donor (host + up to 2 donors) chimerism from allele read counts at any set of informative biallelic loci in standard VCF files. allomix uses maximum likelihood estimation with a sequencing error model adapted from the forensic mixture deconvolution literature, simplified for the clinical transplant setting where donor and host genotypes are known. The tool provides profile likelihood confidence intervals, optional per-marker amplification bias correction, and integrated quality control including multi-timepoint timeline reporting. In silico validation across sequencing depths from 50x to 1,000x demonstrated <1% mean absolute error at all depths tested, ranging from {{ depth_1000.mean_abs_error_pct | dp(2) }}% at 1,000x to {{ depth_50.mean_abs_error_pct | dp(2) }}% at 50x depth. Multi-donor estimation was validated in silico with sibling donor pairs, achieving <1% per-donor MAE. By working with markers already present in clinical workflows, allomix enables laboratories to add chimerism monitoring to existing NGS assays without additional reagents, library preparation, or proprietary software.
