# allomix: donor chimerism monitoring from existing clinical NGS panels

## Abstract

Many clinical laboratories already run targeted next-generation sequencing (NGS) panels that include biallelic polymorphic markers for sample identification or quality control purposes. We reasoned that these markers could be repurposed for quantitative donor chimerism monitoring after allogeneic hematopoietic stem cell transplantation (HSCT), avoiding the need for a separate dedicated assay. We present allomix, an open-source tool that estimates donor DNA fraction from allele read counts at any set of informative biallelic loci in standard VCF files. allomix uses maximum likelihood estimation with a sequencing error model adapted from the forensic mixture deconvolution literature, simplified for the clinical transplant setting where donor and host genotypes are known. The tool provides profile likelihood confidence intervals, optional per-marker amplification bias correction, and integrated quality control. By working with markers already present in clinical workflows, allomix enables laboratories to add chimerism monitoring to existing NGS assays without additional reagents, library preparation, or proprietary software.
