# Empirical Results

Data in this directory was generated by `scripts/measure_panel_bias.py`, which measures per-marker panel characteristics from joint-called genotyping VCFs.

## How it was generated

1. A file listing VCF paths was created from production genotyping runs on `/tau`:

   ```bash
   find /tau/data/clinical_hg38/idt_rhampseq_sid/ -path '*/2_variants/*.gatk.hg38.vcf.gz' \
       -not -path '*/gatk_per_sample/*' > vcf_list.txt
   ```

2. The script was run against that list:

   ```bash
   python scripts/measure_panel_bias.py vcf_list.txt --output paper/empirical_results/panel
   ```

## What the script measures

For each bi-allelic marker across all samples in the input VCFs:

- **Amplification bias** — het VAF deviation from 0.5
- **Locus dropout rate** — no-call (`./.`) frequency
- **Per-marker call rate**
- **Depth distribution** per marker and across the panel
- **Het/hom ratio vs HWE expectation** — allele dropout signal

Default quality filters: `--min-dp 100`, `--min-gq 20`.

## Output files

- `panel_empirical.csv` — panel-wide summary statistics (one row)
- `panel_per_marker.tsv` — per-marker statistics (one row per locus)

Both files contain only summary statistics with no patient identifiers or genomic coordinates.
